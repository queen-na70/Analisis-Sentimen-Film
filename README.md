# Movie Review Sentiment Analysis Using Naïve Bayes

## Description

This project performs sentiment analysis on English movie reviews using the Multinomial Naïve Bayes algorithm with TF-IDF feature extraction. The model classifies movie reviews into Positive or Negative sentiment.

---

## Dataset

IMDb Dataset of 50K Movie Reviews

- Total Reviews: 50,000
- Positive Reviews: 25,000
- Negative Reviews: 25,000

---

## Technologies

- Python
- Pandas
- Scikit-learn
- NLTK
- Streamlit
- Joblib

---

## Machine Learning Workflow

1. Data Collection
2. Data Preprocessing
3. TF-IDF Feature Extraction
4. Train-Test Split
5. Model Training using Multinomial Naïve Bayes
6. Model Evaluation
7. Deployment using Streamlit

---

## Model Performance

Accuracy: **84.88%**

---

streamlit run app/app.py
